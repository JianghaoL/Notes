# MTEC 446A - Computer Assisted Recording



## 2026.1.26 Pro Tools Basics

`T` to zoom in, `R` to zoom out

`Track -> Create Click Track` to create click track.

`CMD + =` Open mixer window

`Option + (Mouse Hover)` Preview audio track

`CMD + Shift + L` Loop audio clip

`Option + (Mouse Drag)` Duplicate the clip at the end
`CMD + D` Duplicate audio clips

`Up / Down Arrow` Snap to grid



`P` to navigate *up* through tracks, `;` to navigate down

`Control + CMD + Up Arrow` Focus on one track
`Control + CMD + Option + Down Arrow` Back to all track
`Option + A` Overview all track (left to right)


`Tab to Transient` + `A` Trim the silence before the triansient

## 2026.2.2 Pro Tools for Voice Over

`CMD + Shift + N` Creat new tracks
